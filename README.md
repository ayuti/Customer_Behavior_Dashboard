# Customer_Behavior_Dashboard
📌 Project Overview
This project represents a complete, industry-standard, end-to-end data analytics workflow. It mirrors the responsibilities of professional analysts by translating raw data into strategic business intelligence through data preparation, modeling, and visualization.

🚀 Workflow Stages
✅ Data Preparation & EDA (Python)
Cleaned and transformed the raw dataset for analysis.

Conducted Exploratory Data Analysis to identify key patterns and outliers.

✅ Data Analysis (SQL)
Simulated business transactions to extract deep insights.

Ran queries to analyze customer segments, loyalty, and primary purchase drivers.

✅ Visualization & Insights (Power BI)
Built an interactive dashboard to highlight trends.

Enabled data-driven decision-making through visual reporting.

🛠️ Tech Stack
Python: Data Cleaning & Transformation

SQL: Business Logic & Data Extraction

Power BI: Interactive Dashboards & Reporting
